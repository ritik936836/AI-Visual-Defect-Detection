# AI Visual Defect Detection

This project uses Deep Learning (TensorFlow + Flask) to detect whether a product image contains a defect.

## Technologies
- Python
- TensorFlow
- Keras
- Flask
- HTML
- CSS

## Features
- Upload Image
- Detect GOOD or DEFECT
- Web Interface
- Deep Learning Model

## Run

pip install -r requirements.txt

python app.py

Open:

http://127.0.0.1:5000
